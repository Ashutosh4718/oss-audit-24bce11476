## Project Overview
This repository contains the practical components for the **Open Source Software Capstone Project**. It includes a set of Bash scripts designed to analyze the Python environment on a Linux system, demonstrating key concepts such as transparency, accessibility, and security in open-source software.

This audit focuses on how the open-source nature of Python has enabled the rapid growth of the modern AI ecosystem.

## Included Scripts
The following scripts were developed to perform a deep-dive audit of the local environment:

1. **`script1.sh`**: Displays system information such as OS details, kernel version, current user, and system uptime.
2. **`script2.sh`**: Checks whether Python 3 is installed on the system and retrieves its version and basic details.
3. **`script3.sh`**: Examines important system directories and reports their disk usage along with permissions and ownership.
4. **`script4.sh`**: Analyzes a system log file (`/var/log/dpkg.log`) to count occurrences of specific keywords related to installations or updates.
5. **`script5.sh`**: An interactive script that generates a personalized open-source manifesto based on user input.

## Conclusion
This project demonstrates a practical understanding of open-source principles, Linux system operations, and shell scripting. It also shows how Python, as an open-source language, plays a significant role in modern computing and technological innovation.
